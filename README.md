# CANSAT_PROJECT_2026

Proyecto de prueba para **visión por cámara** orientado a un sistema CANSAT. El repositorio contiene scripts en **Python** para captura y/o procesamiento de imagen, y utilidades asociadas para experimentar con técnicas de visión computacional.

## Objetivo
Validar un flujo básico de visión (captura → procesamiento → salida/registro) para su integración en un prototipo CANSAT.

## Contenido del repositorio
- **Scripts principales**: ejecución del pipeline de cámara/visión.
- **Módulos auxiliares**: funciones de apoyo (procesamiento, utilidades, etc.).
- **Recursos** (si aplica): configuraciones, ejemplos y/o datos de prueba.

> Nota: esta sección se ajustará con los nombres reales de carpetas y archivos del proyecto.
